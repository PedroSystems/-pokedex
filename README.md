# 🎯 Pokédex Completa

Uma enciclopédia Pokémon interativa e moderna construída com HTML, CSS, JavaScript e a PokeAPI.

![Pokédex](https://img.shields.io/badge/status-funcionando-brightgreen)
![Versão](https://img.shields.io/badge/versão-1.0.0-blue)
![Licença](https://img.shields.io/badge/licença-MIT-green)

## 📋 Funcionalidades

### Busca e Navegação
- 🔍 **Pesquisa** por nome ou número
- 🏷️ **Filtro por tipo** de Pokémon
- 📊 **Ordenação** por número, nome, peso ou altura
- 📄 **Carregamento infinito** com botão "Carregar Mais"
- 🎯 **Resultados em tempo real**

### Detalhes dos Pokémon
- 🖼️ **Imagem oficial** em alta resolução
- 📝 **Nome e número** com formatação
- 🎨 **Tipos** com cores exclusivas
- 📊 **Status base** com barras de progresso
- ⚡ **Habilidades** listadas
- 🔄 **Cadeia de evolução** interativa

### Favoritos
- ⭐ **Adicionar/Remover** favoritos
- 📋 **Lista exclusiva** de favoritos
- 💾 **Persistência** no LocalStorage
- 🔢 **Contador** de favoritos

### Experiência do Usuário
- 🌓 **Modo Escuro/Claro** com persistência
- ✨ **Animações suaves** e transições
- 📱 **Design Responsivo** para todos os dispositivos
- ⌨️ **Atalhos de teclado** (Ctrl+F, ESC)
- 🎯 **Feedback visual** para ações

### Performance
- 🚀 **Carregamento otimizado** com lazy loading
- 💾 **Cache inteligente** de dados
- ⚡ **Renderização eficiente** dos cards
- 📦 **Requisições paralelas** para melhor performance

## 📦 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Design moderno com variáveis CSS
- **JavaScript (ES6+)** - Lógica completa orientada a objetos
- **PokeAPI** - Dados oficiais dos Pokémon
- **Font Awesome** - Ícones profissionais
- **Google Fonts** - Tipografia moderna (Inter)
- **LocalStorage** - Persistência de dados
